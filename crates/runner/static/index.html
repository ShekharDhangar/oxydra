Runner Configuration

Set the directory where Oxydra stores agent workspaces and session data. This can be an absolute path or a relative path from the config directory.

Register First User

Choose the user name you want to use. Oxydra will keep the config file under users/<name>.toml, and on a fresh install it will rename the built-in alice user and file for you automatically.

User config file:

The existing built-in file users/alice.toml will be renamed automatically when you pick a different user name here.

Provider Setup

Connect an AI provider. The Provider Name is your custom identifier (e.g. "google-aistudio"). The Provider Type is the API protocol. Selecting a type auto-fills the other fields.
